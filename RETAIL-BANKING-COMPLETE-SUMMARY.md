## ğŸ¨ Data Architecture Highlights

### Bronze Layer (Raw Zone)
- **Purpose:** Immutable, raw data from source systems
- **Format:** Schema-on-read, JSON, Avro, Parquet
- **Processing:** CDC (Change Data Capture), Streaming ingestion
- **Governance:** Full audit trail, source system lineage

### Silver Layer (Curated Zone)
- **Purpose:** Cleaned, conformed, golden records
- **Features:**
  - Master Data Management (MDM)
  - Slowly Changing Dimensions (SCD Type 2)
  - Data quality scoring (completeness, accuracy, consistency, timeliness)
  - Deduplication & survivorship rules
- **Processing:** Batch processing (hourly/daily)

### Gold Layer (Consumption Zone)
- **Purpose:** Analytics-ready dimensional model
- **Methodology:** Kimball Dimensional Modeling
- **Features:**
  - Star schemas optimized for BI
  - Conformed dimensions across domains
  - Fact tables with transaction & snapshot grains
  - Pre-aggregated metrics
- **Consumption:** Power BI, Tableau, Python/R, ML models
